# Frontend DAC

Frontend React/Vite du MVP CodeCamp ETNA 2026.

## Demarrage local

Depuis `frontend/`:

```bash
npm install
npm run dev
```

Variable principale:

```bash
VITE_API_URL=http://localhost:8000
```

## Pages actives

- Landing
- Auth login/register
- Onboarding AWS
- Dashboard
- Chat DAC
- Resources
- Profile/Settings

Les pages experimentales ou non montees ont ete archivees dans `archive/legacy-frontend/`.
